This etc directory contains config files used bt the seaside/bin scripts

DO NOT MOVE THIS DIRECTORY, IT IS USED BY COMMANDS SHIPPED WITH
YOUR GEMSTONE SYSTEM. However you can modify the files to reflect your
GemStone configuration.

The files include:
    gemstone.conf    - environment varialbles, including location of this installation
    gemsone.secret   - the system passwords, so they won't show up in the command line
    gemstone.key     - a keyfile which enables the system to run
    gemstone.version - the SVN build version
